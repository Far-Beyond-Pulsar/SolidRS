# Implementing a Saver

This guide walks through writing a complete [`Saver`] for a format crate.
Savers serialise a [`Scene`] to a byte stream; they mirror the [`Loader`]
implementation pattern.

---

## 1. Define `FormatInfo`

```rust
use solid_rs::traits::FormatInfo;

static MY_FORMAT: FormatInfo = FormatInfo {
    name:         "My Format",
    id:           "myfmt",
    extensions:   &["myfmt"],
    mime_types:   &["model/x-myfmt"],
    can_load:     false,
    can_save:     true,
    spec_version: Some("1.0"),
};
```

---

## 2. Implement the `Saver` Trait

```rust
use solid_rs::prelude::*;
use solid_rs::traits::{SaveOptions, Saver};
use std::io::Write;

pub struct MyFmtSaver;

impl Saver for MyFmtSaver {
    fn save(
        &self,
        scene: &Scene,
        writer: &mut dyn Write,
        options: &SaveOptions,
    ) -> Result<()> {
        let mut w = std::io::BufWriter::new(writer);
        write_scene(scene, &mut w, options)
    }

    fn format_info(&self) -> &FormatInfo {
        &MY_FORMAT
    }
}
```

---

## 3. Direct Iteration

The simplest approach iterates over scene arrays directly:

```rust
fn write_scene(
    scene: &Scene,
    w: &mut dyn Write,
    opts: &SaveOptions,
) -> Result<()> {
    // Header
    if let Some(copyright) = &opts.copyright {
        writeln!(w, "# Copyright: {copyright}").map_err(SolidError::Io)?;
    }
    let gen = opts.generator.as_deref().unwrap_or("solid-rs");
    writeln!(w, "# Generated by {gen}").map_err(SolidError::Io)?;

    // Materials
    for (i, mat) in scene.materials.iter().enumerate() {
        writeln!(w, "mtl {i} {}", mat.name).map_err(SolidError::Io)?;
    }

    // Meshes
    for mesh in &scene.meshes {
        writeln!(w, "mesh {}", mesh.name).map_err(SolidError::Io)?;
        for v in &mesh.vertices {
            writeln!(w, "v {} {} {}", v.position.x, v.position.y, v.position.z)
                .map_err(SolidError::Io)?;
        }
        for prim in &mesh.primitives {
            let mat = prim.material_index
                .map(|i| i.to_string())
                .unwrap_or_else(|| "-1".into());
            write!(w, "f {mat}").map_err(SolidError::Io)?;
            for &idx in &prim.indices {
                write!(w, " {idx}").map_err(SolidError::Io)?;
            }
            writeln!(w).map_err(SolidError::Io)?;
        }
    }
    Ok(())
}
```

## 4. Visitor-Based Approach

For large or streaming formats, implement [`SceneVisitor`] instead:

```rust
// The visitor borrows the writer for its lifetime.
struct MyFmtWriter<'a> {
    writer:  &'a mut dyn Write,
    options: SaveOptions,
}

impl SceneVisitor for MyFmtWriter<'_> {
    fn visit_mesh(&mut self, mesh: &Mesh, _index: usize) -> Result<()> {
        writeln!(self.writer, "mesh {}", mesh.name)
            .map_err(SolidError::Io)?;
        for v in &mesh.vertices {
            writeln!(self.writer, "v {} {} {}", v.position.x, v.position.y, v.position.z)
                .map_err(SolidError::Io)?;
        }
        Ok(())
    }

    fn visit_material(&mut self, mat: &Material, index: usize) -> Result<()> {
        let c = mat.base_color_factor;
        writeln!(self.writer, "mtl {index} {} {} {} {}",
            c.x, c.y, c.z, c.w).map_err(SolidError::Io)
    }
}

impl Saver for MyFmtSaver {
    fn save(&self, scene: &Scene, writer: &mut dyn Write, opts: &SaveOptions) -> Result<()> {
        let mut v = MyFmtWriter { writer, options: opts.clone() };
        scene.visit(&mut v)
    }
    fn format_info(&self) -> &FormatInfo { &MY_FORMAT }
}
```

## 5. Reading Format-Specific Extensions

Savers can read extension data that loaders attached earlier:

```rust
use solid_myfmt::MyFmtMeshMeta;

fn write_mesh(mesh: &Mesh, w: &mut dyn Write) -> Result<()> {
    let id = mesh.extensions
        .get::<MyFmtMeshMeta>()
        .map(|m| m.object_id)
        .unwrap_or(0);

    writeln!(w, "mesh_id {id}").map_err(SolidError::Io)?;
    // … rest of mesh output …
    Ok(())
}
```

---

## 6. Register the Saver

```rust
registry.register_saver(solid_myfmt::MyFmtSaver);

let opts = SaveOptions {
    embed_textures: true,
    ..Default::default()
};
registry.save_file_with_options(&scene, "output.myfmt", &opts)?;
```

---

## Checklist

- [ ] `FormatInfo` has correct `extensions` and `mime_types`.
- [ ] All `io::Error` conversions use `SolidError::Io(e)`.
- [ ] `SaveOptions::copyright` and `generator` are written to the file header.
- [ ] `SaveOptions::embed_textures` is honoured where the format supports it.
- [ ] Format-specific metadata is read from `extensions` when present.
- [ ] Public API items have doc comments.
